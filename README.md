# trainee-db-notes

This repo contains short notes about:

**Repository (Repo):**
  A project folder on GitHub that stores your files and the full history of changes. Used to organize and share work.
**Clone:**
   Create a local copy of a GitHub repository on your computer so you can work on it offline.
**Commit:**
  A saved snapshot of your changes with a short message describing what you updated. Helps track progress and history.
**Push:**
  Upload your local commits to GitHub so the remote repository is updated and others can see your changes.
**Pull:**
   Download the latest changes from GitHub to your local copy. Do this before starting work to stay up to date.
**Branch:**
   A separate line of work (like a copy of the project) to build a feature without affecting the main branch.
**Merge:**
  Combine changes from one branch into another after the work is ready.
**Pull Request (PR):**
  A request to merge your branch into `main`. It enables review and discussion before merging.
